# QA Automation Engineer Portfolio

Portfolio profesional empresarial desarrollado con:

- React 19
- TypeScript
- Vite
- NestJS 11
- Prisma
- MySQL
- Docker
- GitHub Actions
- Three.js
- React Three Fiber

---

# Requisitos

- Node.js 22 LTS
- pnpm 10+

---

# Instalación

## Clonar repositorio

```bash
git clone <repository-url>
```

## Ingresar al proyecto

```bash
cd portfolio-qa
```

## Instalar dependencias

```bash
pnpm install
```

---

# Frontend

```bash
pnpm --filter frontend dev
```

Frontend:

```text
http://localhost:5173
```

---

# Backend

```bash
pnpm --filter backend start:dev
```

Backend:

```text
http://localhost:3000
```

---

# Build

```bash
pnpm build
```

# Lint

pnpm lint
